# Task Manager Web App

## Overview

This project is a Task Manager Web App created with HTML, CSS, and JavaScript. The purpose of the app is to help users create, organize, and manage tasks in a clear and simple way.

My main goal was to build something functional and intuitive. Users can add tasks, choose a priority level, mark tasks as important or completed, and delete tasks when needed.

## Features

- Add new tasks
- Choose a priority: High, Medium, or Low
- Mark tasks as important
- Mark tasks as completed
- Delete tasks
- Show the date when the task was added
- Move task cards using drag and drop
- Log task updates in the console with `JSON.stringify()`

## Technologies Used

- HTML
- CSS
- JavaScript
